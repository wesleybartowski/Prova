1 - O que é clone?
É clonar o repositorio online para um repositorio local.
2 - O que é commit?
É alterações em um codigo.
3 - O que é um repositório?
É a pasta do seu projeto.
5 - O que é pull?
É uma atualização do repositorio local. ele puxa do repositorio online. como se fosse um merge!
7 - O que é branch?
São ramificações do seu projeto.
8 - O que é pull request?
Ela e um convite para o dono do repositorio. com pedido ou inteção de alterações no seu projeto principal.
9 - O que é fork?
É um clone de um repositorio online. para seu perfil "conta" online.
